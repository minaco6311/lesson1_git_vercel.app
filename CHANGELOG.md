# CHANGELOG

## [2026-06-24] クイズバトルに NEXT ボタンを追加・UI 微調整

### 変更内容
- クイズ回答後の自動送り（7秒タイマー）を廃止し、「▶ NEXT」ボタンで次の問題に進む方式に変更
- 解説ボックスの下に NEXT ボタンを表示、次の問題開始時に自動で消える
- 吹き出しの表示位置を下に調整（`top: -8px` → `top: 20px`）
- メインコンテンツエリアの上部余白を削減（`padding: 20px` → `padding: 12px`）

### 背景・理由
- 解説がすぐ消えてしまい読み切れないという問題を解消するため
- 自分のペースで次の問題に進めるほうが学習効果が高いため

## [2026-06-24] GIT QUEST 初期公開

### 変更内容
- `index.html` — Git 学習サイト「GIT QUEST」を新規作成
  - コマンド道場：主要 Git コマンドをクリックで解説・サンプル表示
  - クイズバトル：RPG バトル形式の4択クイズ（HP・スコア・レベル管理）
  - Git の地図：ワーキングツリー→ステージ→ローカル→リモートの流れを可視化
  - 馬キャラクターのマスコット（`mascot.png`）と吹き出しアニメーション（ぽよーん）
  - ピクセルアートテーマ（Press Start 2P フォント・ドット絵風 UI）
  - Canvas 2D APIによる星空背景アニメーション
  - Web Audio API による 8-bit 効果音
- `mascot.png` — ホッキョクグマのマスコット画像を追加
- `CLAUDE.md` — CHANGELOG 記録ルールを定義

### 背景・理由
- Git 初心者がコマンドを楽しく学べるインタラクティブなサイトを作るため
- スクロール不要で主要コンテンツが見えるコンパクトな UI を目指した
